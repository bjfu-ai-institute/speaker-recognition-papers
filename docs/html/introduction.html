Introduction¶

Python-ASV is an open source package for speaker recognition. The aim of this project is share the implementation of model in the recent and classic paper. We want to provide a easy-to-use toolkit for later research and education. We may sacrifice performance to increase readability because we won’t support large-scale deployments.

Warning

This project is done in authors’ spare time, and the code may be wrong. Early publication is for better collaborative development. It is not recommended to use this repo on important projects now.

git:
